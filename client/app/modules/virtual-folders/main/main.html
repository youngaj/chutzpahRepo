Welcome to the new and improved Virtual Folders!


Detailed instructions and feature overviews are pending.
  • To get started. Please select an item on the left.
  • To expand a tree or folder select the horizontal triangle to the left of the item.
  • To see item details click the icon and/or item title. Detailed information will display in the "info" section on the right..
  • To create a new tree click the "Add Tree" button on the bottom of the Virtual Folders panel
  • To create a new folder, shared file, photo or link right click on a select the appropriate "Insert" selection.
  • To edit your security settings select the "Security" Tab then click the "Security Wizard" button.
  • To view/edit/attach files select the "File Container" Tab.
  • To add or view comments select the "Comment" Tab.
  • To get notifications about updates select the "Follower" Tab.
